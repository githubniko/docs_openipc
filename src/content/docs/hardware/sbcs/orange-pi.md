Use this page as the Orange Pi entry point for OpenIPC SBC ground-station builds.

## Firmware Images

Use the [latest OpenIPC SBC Ground Station release page](https://github.com/OpenIPC/sbc-groundstations/releases/tag/latest) as the entry point. For Orange Pi 3B, Orange Pi 5, and Orange Pi 5 Plus images, follow the Orange Pi release linked there.

The Orange Pi release has provided image names such as:

- `OrangePi3b_openIPC.img.xz`
- `OrangePi5_OpenIPC.img.xz`
- `OrangePi5plus_OpenIPC.img.xz`

Check the release page before flashing because filenames, supported boards, and image tracks can change.

## Use Cases

Orange Pi boards are relevant to ground-station and video receiver work, especially where Rockchip acceleration or a larger SBC is preferred over a small receiver board.
