---
title: Обновления прошивки
description: Как обновлять камеру с установленной OpenIPC.
---

Эта страница относится к устройствам, на которых OpenIPC уже установлена и
камера нормально загружается. Для первой установки или восстановления используйте
инструкцию именно для вашего устройства или SoC.

## Перед обновлением

- Убедитесь, что устройство уже работает на OpenIPC.
- Обеспечьте стабильное питание на все время обновления.
- Запишите текущий firmware target и версию.
- Сохраните важную пользовательскую конфигурацию.
- Убедитесь, что камера может обращаться к GitHub, если планируете обновление
  через кнопку Web UI.

## Обновление через Web UI

1. Откройте Web UI камеры в браузере.
2. Войдите в интерфейс, если он запросит учетные данные.
3. Откройте **Settings -> Firmware** или **Firmware -> Update**.
4. Выберите **Install update from GitHub**.
5. Дождитесь завершения обновления и автоматической перезагрузки.

Обновление может занять несколько минут. Не отключайте питание камеры, пока
прошивка записывается.

## После обновления

- Обновите страницу Web UI и проверьте отображаемую версию прошивки.
- Проверьте видеопоток и нужные интеграции.
- Верните пользовательские настройки только если они не сохранились.
- Если камера не загружается, используйте способ восстановления, описанный для
  вашего оборудования.
