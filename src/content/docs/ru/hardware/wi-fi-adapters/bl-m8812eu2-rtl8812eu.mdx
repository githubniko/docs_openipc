RTL8812EU - новый, недорогой и широко используемый Wi-Fi чип, который встречается во множестве Wi-Fi адаптеров. Этот чип был выпущен относительно недавно, поэтому возможности driver injection/monitor пока ограничены: сейчас поддерживается передача в режимах 5MHz, 10MHz и 20MHz, а 40MHz доступен только на приём. Это лучший выбор для прошивки APFPV. На этой странице собраны сведения об известных Wi-Fi адаптерах с этим чипом.

:::caution
Если после включения режима 40MHz изображение не принимается, прочитайте эту [статью](/ru/use-cases/fpv/wfb-ng/unbrick-eu-bu/), чтобы исправить проблему.
:::

## BL-M8812EU2

LB-LINK BL-M8812EU2 - недорогой и мощный Wi-Fi адаптер, который можно встретить в [Runcam WiFiLink 1](/ru/hardware/air-units/runcam/runcam-wifilink1/).

### Распиновка

| Номер пина | Функция |
|:----------:|:-------:|

| 6 | Вход питания 5V |
| 7 | Вход питания 5V |
| 8 | GND |

### Характеристики

| Host interface | USB 2.0 |
|:--------------:|:-------:|
| Power Supply | DC 5.0V +-0.25V @ 1800mA (MAX) |
| Antenna connection | IPEX x2 |
| Power output | &lt;=29dBm(&lt;=800mW) |
| Supported modes | 5MHz, 10MHz, 20MHz, 40MHz (только приём) |
