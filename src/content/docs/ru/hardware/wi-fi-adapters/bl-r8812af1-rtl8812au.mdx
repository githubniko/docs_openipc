RTL8812AU - старый, дорогой, но проверенный Wi-Fi чип, который встречается во многих сетевых картах. NIC на базе AU имеют отличную поддержку сообщества, поэтому режимы injection/monitor 5Mhz, 10MHz, 20MHz и 40MHz могут принимать и передавать. Поэтому оборудование на базе AU может достигать выдающихся битрейтов: например, 21-04-2025 суммарно было достигнуто 52Mbps, из них 36Mbps видео (WFB-NG). На этой странице собраны сведения об известных Wi-Fi адаптерах с этим чипом.

:::caution
Драйвер RTL8812AU сообщает сглаженный RSSI, который реагирует очень медленно, поэтому прошивка APFPV на базе `hostapd` даёт на этом Wi-Fi чипе дерганую картинку.
:::

## BL-R8812AF1

LB-LINK BL-R8812AF1 - маленькая сетeвая карта с низким энергопотреблением. Это очень популярный выбор для toothpick-сборок. Эта карта встречается в EMAX Wyvern Link VTX Alpha 100mW.

:::caution
Эта сетeвая карта может очень быстро сильно нагреваться, поэтому лучше сделать для неё какой-либо радиатор.
:::

### Распиновка

| Номер пина | Функция |
|:----------:|:-------:|
| 1 | PDN |
| 2 | VDD(3.3V) |
| 3 | DM- |
| 4 | DP+ |
| 5 | GND |
| 6 | LED |

### Характеристики

| Host interface | USB 2.0 |
|:--------------:|:-------:|
| Power Supply | DC 3.3V +-10% |
| Antenna connection | IPEX x2 |
| Power output | &lt;=20dBm(&lt;=100mW) |
| Supported modes | 5MHz, 10MHz, 20MHz, 40MHz |

## Сетевая карта из набора WiFiLink-G

Это сетeвая карта, которую можно получить при покупке набора WiFiLink-G. У неё низкое энергопотребление, поэтому она популярна для использования с Android GS.

![image](/images/rtl8812au-wifilink-g-adapter.png)

### Характеристики

| Host interface | USB 3.0 |
|:--------------:|:-------:|
| Power Supply | DC 5V |
| Antenna connection | RP-SMA x2 |
| Power output | NA |
| Supported modes | 5MHz, 10MHz, 20MHz, 40MHz |
