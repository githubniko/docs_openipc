AP FPV uses a regular Wi-Fi access point style link rather than WFB-NG. Start with the preserved beginner guide, then choose a firmware image that matches the exact target board and flash layout.

## Firmware Image

The related FPV resources page links a ready-made [latest AP FPV image for SSC338Q NOR](https://github.com/OpenIPC/builder/releases/download/latest/openipc.ssc338q-nor-apfpv.tgz). That archive is specific to SSC338Q NOR targets; do not use it for a different SoC or flash layout.

For other devices, check [OpenIPC Builder latest releases](https://github.com/OpenIPC/builder/releases/tag/latest) or build a matching image with [OpenIPC Builder](/software-tools/openipc-builder/).
